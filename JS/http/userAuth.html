# 确认访问用户身份的认证

# 认证

  • 密码：只有本人才会知道的字符串信息
  • 动态令牌：仅限本人持有的设备内显示的一次性密码
  • 数字证书：仅限本人（终端）持有的信息
  • 生物认证：指纹和虹膜等本人的生理信息
  • IC卡等：仅限本人持有的信息

# HTTP使用的认证方式

  • BASIC认证（基本认证）
  • DIGEST 认证（摘要认证）
  • SSL客户端认证
  • FormBase 认证（基于表单认证）
